== xbps-freedom

(unofficial) XBPS tool to list and count packages that respect your freedom. +
Inspired by link:https://github.com/vmavromatis/absolutely-proprietary[vmavromatis' project].

=== Installing

[source,bash]
----
sh install.sh
----
